SITE MONITORING APP

Open index.html to test on a computer.
Default PIN: 1234. Change it in app.js.

Important: this version stores entries on the device/browser using localStorage. It is perfect for testing the interface. For multiple users sharing the same live database, connect it to Firebase or Google Sheets.

To make it available on phones for free:
1. Create a free GitHub account.
2. Create a new repository called site-monitoring-app.
3. Upload these files.
4. Go to Settings > Pages.
5. Set source to main branch/root.
6. Open the Pages link on a phone.
7. Add it to the home screen.
